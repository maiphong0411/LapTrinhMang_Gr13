# Sổ tay sinh viên

## Pre-requisites
The project was developed using python 3.8.5 with the following packages.
- Anytree
- Pyautogui
- Streamlit
- Socket
- Pickle

Installation with pip:
```bash
pip install -r requirements.txt
```

## Getting Started
Run server ```bash streamlit run app.py```
Run client ```bash streamlit run app.py```
